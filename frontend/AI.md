# Installation

## Clone Repository

```bash
git clone https://github.com/PadlanPutra-dev/Heart-Guardian-2026.git
```

## Backend

```bash
cd backend
npm install
npm start
```

## Frontend

```bash
cd frontend
npm install
npm start
```

## Database

Install MySQL locally and configure the `.env` file inside the backend directory.

Example:

```env
DB_HOST=localhost
DB_USER=root
DB_PASSWORD=your_password
DB_NAME=heart_guardian
```
