# Heart Guardian Backend

## Setup

1. Install dependencies:
   npm install

2. Start PostgreSQL locally or via Docker.

3. Create a database named `heart_guardian`.

4. Copy `.env.example` to `.env` and adjust credentials if needed.

5. Start the server:
   npm run dev

## API Endpoints

- GET /health
- POST /api/auth/register
- POST /api/auth/login
- GET /api/me
